**Mapa de Vida** es una herramienta digital de autoconocimiento basada en la numerología de base 22 y los arquetipos de Alejandro Lavín. Esta aplicación permite a los usuarios decodificar su propósito de vida, talentos innatos y desafíos evolutivos a través de una interfaz accesible, inclusiva y multilingüe.

---

## 🚀 Características Principales

- **Pentagrama Visual Dinámico:** Visualización geométrica de los 5 pilares: Alma, Karma, Regalo, Destino y Misión.
- **Análisis Profundo de Arquetipos:** Descripciones detalladas que incluyen esencia, sombra, misión y tareas de activación.
- **Accesibilidad (A11y):** - Soporte para **Alto Contraste** (diseñado para usuarios con baja visión o daltonismo).
  - Estructura semántica compatible con lectores de pantalla.
  - Diseño responsivo "Mobile-First".
- **Multilingüe:** Disponible en Español, Inglés y Portugués.
- **Exportación a PDF:** Generación de reportes personalizados para guardar o imprimir.
- **Social Sharing:** Integración nativa para compartir resultados vía WhatsApp y redes sociales.

---

## 🛠️ Tecnologías Utilizadas

Para este proyecto se priorizó la ligereza y el rendimiento (Zero Dependencies en el core):

* **HTML5 & CSS3:** Uso de variables CSS para el sistema de temas (Alto Contraste) y Flexbox/Grid para el diseño responsivo.
* **JavaScript (Vanilla):** Lógica de cálculo numerológico y manipulación del DOM sin frameworks pesados.
* **SVG:** Para la renderización del Pentagrama geométrico.
* **html2pdf.js:** Librería externa para la conversión de reportes a PDF.
